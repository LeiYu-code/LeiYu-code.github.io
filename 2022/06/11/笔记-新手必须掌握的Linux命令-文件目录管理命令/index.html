笔记-新手必须掌握的Linux命令-文件目录管理命令

[南月晨的blog](南月晨的博客 (leiyu-code.github.io))

leiyu-code.github.io

Linux命令大全(手册) – 真正好用的Linux命令在线查询网站 (linuxcool.com)

touch命令

touch [参数] 文件名称

用于创建空白文件或设置文件的时间

参数 作用
-a 仅修改“读取时间”（atime）
-m 仅修改“修改时间”（mtime）
-d 同时修改atime与mtime

mkdir命令

cp [参数] 源文件名称 目标文件名称

用于复制文件或目录，英文全称为“copy”

如果目标文件是目录，则会把源文件复制到该目录中；

如果目标文件也是普通文件，则会询问是否要覆盖它；

如果目标文件不存在，则执行正常的复制操作。

参数 作用
-p 保留原始文件的属性
-d 若对象为“链接文件”，则保留该“链接文件”的属性
-r 递归持续复制（用于目录）
-i 若目标文件存在则询问是否覆盖
-a 相当于-pdr（p、d、r为上述参数）

mv命令

rm [参数] 文件 名称

用于删除文件或目录，英文全称为“remove”

参数 作用
-f 强制执行
-i 删除前询问
-r 删除目录
-v 显示过程

dd命令

dd if=参数值of=参数值count=参数值bs=参数值

用于按照指定大小和个数的数据块来复制文件或转换文件

参数 作用
if 输入的文件名称
of 输出的文件名称
bs 设置每个“块”的大小
count 设置要复制“块”的个数

file命令

file文件名称

用于查看文件的类型

Tips

在Windows系统中打开文件时，一般是通过用户双击鼠标完成的，系统会自行判断用户双击的文件是什么类型，因此需要有后缀进行区别。而Linux系统则是根据用户执行的命令来调用文件，例如执行cat命令查看文本，执行bash命令执行脚本等，所以也就不需要强制让用户给文件设置后缀了。

tar命令

tar参数 文件名称

用于对文件进行打包压缩或解压

参数 作用
-c 创建压缩文件
-x 解开压缩文件
-t 查看压缩包内有哪些文件
-z 用Gzip压缩或解压
-j 用bzip2压缩或解压
-v 显示压缩或解压的过程
-f 目标文件名
-p 保留原始的权限与属性
-P 使用绝对路径来压缩
-C 指定解压到的目录
